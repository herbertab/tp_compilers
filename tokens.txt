*Enumeracao dos tokens da linguagem L

1- palavras reservadas 
2- atribuição (=)
3- tipos de dados (string boolean byte integer)
4- abre parenteses ( '(' )
5- fecha parenteses ( ')' )
6 - operador relacional ( > < <> >= <= ==)
7- pontuação (, ;)
8 - operador aritmético (+ - * /)
9- abre chaves ({)
10- fecha chaves (})
11- constante
12- variaveis
